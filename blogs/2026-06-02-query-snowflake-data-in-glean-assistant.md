Glean has announced that Snowflake is now available as a native data source in Glean Assistant, allowing employees to query Snowflake data using natural language without requiring SQL expertise. The company was named the 2026 AMER Snowflake Product Innovation Partner of the Year for enabling any employee to query Snowflake data directly from Glean Assistant with no agent to build, no tool to switch, and no SQL required. The integration combines Snowflake's text-to-SQL capabilities with Glean's enterprise context to provide answers grounded in both structured data and unstructured company knowledge.
